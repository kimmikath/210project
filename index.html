ArcScene is a 3D visualization application that allows the user to view GIS data in three dimensions. It also serves as a platform to overlay layers of raster and vector data in a 3D environment. Features are presented in 3D by providing height information from feature geometry, feature attributes, layer properties, or a defined 3D surface (Digital Elevation Model). All data should also be projected to a common projection to create a 3D scenery. The sample 3D terrain visualization of Mt. Maculot situated in Municipality of Cuenca, Batangas is shown below. The elevation model (30m) from SRTM data reveals the topography of the mountain. In addition, landslide prone areas were identified and visualized as a precaution for hikers as well as the community.
Virtual Terrain Project aims to develop tools to construct real world scenes in a 3D digital form. It is an open source 3D visualization tool that utilizes GIS data to create 3D environment for planning, education and tourism purposes. VTP includes software such as VT Builder that organizes the GIS data while VTP Enviro renders the data in 3D. The topography of Mt Maculot visualized in VTP is presented below.
Terragen is an open source tool for building, rendering, and animating realistic natural environments. This software import real world terrain data sets and construct the most realistic visualizations possible. It is easier to manipulate the weather, landscape, water bodies and atmospheric effects based on user preference. The topography of Mt Maculot visualized in Terragen is presented below.
